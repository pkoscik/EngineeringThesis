
\chapter{Integrating Dromajo into Renode Framework}

This chapter aims to cover the implementation process of the Dromajo simulator in the Renode Framework.
To provide a reader with better understanding concerning the integration process, this chapter will also cover
the basics of the Renode Framework, the structure of the simualated devices and basics of CPU simulation
implementation in the framework.

\section{Renode basics}

The Renode Framework is an open-source, permisively licenced (MIT licence), embedded device simulation framework.
Beign mostly written in C\#, the software provides organised, eaisly expandable, clean codebase, rich in object
oriented pragmizms. It can be compiled both using \texttt{mono} and \textttt{.NET} build systems.
